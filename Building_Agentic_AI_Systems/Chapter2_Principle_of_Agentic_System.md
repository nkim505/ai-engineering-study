# 자기관리, 주체성, 자율성 이해하기

에이전틱 시스템의 핵심 개념 1. 자기관리 2. 주체성 3. 자율성

1. 자기관리
    1. 자기 조직: 스스로 내부 프로세스, 자원, 행동 조직화 및 구조화
    2. 자기 규제: 스스로의 출력, 행동을 모니터링하고 조정
    3. 자기 적응: 환경 변화, 내부 상태 변화에 따라 행동, 전략, 의사결정 프로세스 수정
    4. 자기 최적화: 학습, 경험, 진화적 프로세스로 스스로의 능력을 지속 향상
    5. 자기 결정: 내부 의사 결정 프러세스를 통해 목표, 우선순위, 실행 방안을 설정
    
2. 주체성 agency
    1. 의사 결정 권한
    2. 의도성: 단순 반응적 시스템이 아닌 목적의식과 추구하는 바가 존재한다는 것
    3. **책임 소재: 주체성을 가진 시스템은 그 결정과 영향에 대한 책임을 가지는 주체로 간주됨
    
    **: 이 부분에 대해서 논란이 있어 보임.
    

1. 자율성
    - 주체나 시스템의 독립성 정도에 초점. 아래처럼 여러 측면으로 나눌 수 있음.
        1. 운용 자율성: 인간 개입, 통제 없이 일련의 작업을 수행할 수 있는 능력
            - 배치 운용을 직접하는?
        2. 기능 자율성: 시스템이 자신의 환경, 콘텍스트에 따라 선택하고 행동하는 능력
            - mcp tool 선택하고 이런 정도?
        3. 계층적 자율성: 높은 - 하위 시스템, 조직 운영 전반에 영향을 미칠 수 있는 결정 폭 넓게 내림. 낮은 - 상위 시스템이나 주체로부터 더 많은 제약이나 감독 받음.
    - AI 시스템에서 자율성은 인간의 감독이나 통제의 완전한 부재는 아님. 일반적으로 설계나의 설정과 제약 내에서 작동하면서 자율성 발휘. 프로그래밍된 경계 내에서 자율적으로 작동.

# 에이전틱 시스템 아키텍쳐 탐색

대표적인 아키텍쳐 패턴

1. 계획 기반형
2. 반응형
3. 하이브리드

## 1. 계획 기반형 아키텍쳐

- `지식기반/기호기반 아키텍쳐`
    - 인지-계획-행동 사이클을 따름
- 장점: 계획 수립, 문제 해결, 의사결정과 같이 복잡한 추론이 필요한 작업 처리 가능
    - 규칙 기반 추론, 제약 만족, 휴리스틱 탐색 등으로 문제 공간 탐색 및 적잘한 전략 수립
    - 명확하게 정의된 환경에서 복잡한 추론, 계획 수립, 의사 결정까지 작업 처리
- 단점: 복잡한 지식 기반 유지하고 그 위에서 추론 수행하는데 들어가는 계산 비용이 높음.
    - 빠른 대응이 요구되는 동적 환경에서는 실시간 대응력 저하 가능
    - 명시적인 지식 표현이 필요해서 지식 형식화가 어렵거나 지속적으로 변하는 도메인에서 적용 어려움
- how to 극복?
    - 하이브리드 아키텍쳐의 반응형이나 행동 기반 요소와 결함 (복잡 추론, 빠른 반응 동시 가능하도록)
- 어느 영역에 주로
    - 로보틱스, 의사결정 지원 시스템, 지능형 투터링 시스템
    - 과 같이 복잡한 의사결정, 계획 수립, 추론이 필수인 영역에서 중요한 구성요소로 활용

## 2. 반응형 아키텍쳐

- `행동기반/자극-반응 아키텍쳐`
    - 환경으로부터 들어오는 자극에 즉각적으로 반응하는데 초점
    - 명시적 모델이나 복잡한 추론 과정 필요로 하지 않음
    - 지각과 행동을 직접 매핑하며 일반적으로 단순한 조건-행동 규칙이나 신경망을 활용함.

반응형 아키텍쳐 구조

- 핵심 속성과 특징
    1. 속도와 반응성
    2. 강건성과 장애 허용성
    3. 불확실성 처리
    4. 병렬 및 분산 처리
    5. 복잡한 행동의 창발
- 한계
    1. 장기적 계획의 부재
    2. 제한된 추론 및 추상화 능력: 주로 낮은 수준의 자극-반응 수준에서 작동
    3. 제한된 학습 능력: 역동적인 환경이나 지속적인 적응이 필요한 환경에는 부적합할 수 있음
    

## 3. 하이브리드 아키텍쳐

계층 구조

- 반응형 계층
    - 신속한 저수준의 반응을 담당.
    - 환경과의 실시간 상호작용을 처리하는 역할
    - 실시간 피드백, 콘텍스트 인식을 계획 기반형 계층에 제공해서 의사결정 프러세스를 지원
- 계획 기반형 계층: 고수준의 추론 및 계획 수립을 담당
    - 추론, 계획, 의사결정 단계를 처리
    - 모델, 목표, 제약 조건등을 유지하면서 추론, 장기적 행동 계획을 가능하게 함.
    - 반응형 계층에 고수준의 계획, 목표, 제약을 제공하여 행동을 유도함.

사용 기술

- 작업 분해
- 복수 계획 선택
- 외부 모듈과의 연계
- 성찰 및 개선
- 메모리 기반 계획
